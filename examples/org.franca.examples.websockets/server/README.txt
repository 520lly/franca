--------------------------------------------------------------------------------------------------

This example Websocket server is implemented in JavaScript based on node.js and websocket.io.

--------------------------------------------------------------------------------------------------

INSTALLATION:
1. Install node.js: Visit http://nodejs.org and click on the Install button to get started.
2. Install websocket.io: Change to this folder and type "npm install websocket.io".

--------------------------------------------------------------------------------------------------

HOW TO RUN IT:
1. Generate the stub's JS code by executing the WebsocketGenTest JUnit test in the Eclipse IDE.
   You need Franca 0.8.11 or later for doing this successfully.
2. In this folder, simply issue "node server.js". This will start the server on port 9000.
3. Open your browser and browse to "" in order to start the client side.
   We have tested this with the Google Chrome, version 30.0.1599.69.
